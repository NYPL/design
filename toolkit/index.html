Forms

Forms fields should be wrapped by a fieldset tag.

Text fields


.my-text-field {
  @include text-fieldset;
}
      

<fieldset class="my-text-field">
  <label for="keyword-text">Title or description</label>
  <input id="keyword-text" type="text">
</fieldset>

Select boxes


.my-select-box {
  @include select-fieldset;
}
      

<fieldset class="my-select-box">
  <label for="topics">Topics</label>
  <select id="topics" name="topics">
    <option value="1">One</option>
    <option value="2">Two</option>
    <option value="3">Three</option>
    <option value="…">…</option>
  </select>
</fieldset>

Alphabetical filters

Not a form element per se but still used within search forms. Selected element is replaced with span. The "Any" selection has a long class.

Any # A B C … 

.alphabetical {
  @include alphabetical-filter;
}

NOTE: Maybe replace a tag with button?


<fieldset class="alphabetical">
  <label>Name starts with</label>
  <span class="long">Any</span>
  <a href="#" title="Show only resources with name starting with a number">#</a>
  <a href="#" title="Show only resources with name starting with A">A</a>
  <a href="#" title="Show only resources with name starting with B">B</a>
  <a href="#" title="Show only resources with name starting with C">C</a>
  <a href="#" title="Show only resources with name starting with …">…</a>
</fieldset>

Radiobutton/checkbox groups

For multiple browser-compatibility reasons we avoid the legend tag and use a span tag with a nested strong tag.

Accessible from 

.radiogroup {
  @include radiobutton-fieldset;
}

NOTE: In this case the input tags are wrapped by the label tag.


<fieldset class="radiogroup">
  <span><strong>Accessible from</strong></span>
  <label for="available-every">
    <input id="available-every" type="radio" name="available" value="any">Anywhere
  </label>
  <label for="available-home">
    <input id="available-home" type="radio" name="available" value="home">Home with a library card
  </label>
  <label for="available-library">
    <input id="available-library" type="radio" name="available" value="library">A library branch
  </label>
</fieldset>

Tables and Data

General tables

Search results summary

Presents a comma-separated list of search terms or filters along with the number of results. The result count has a special span tag (should be number-formatted e.g.: 1,232). Each applied facet or term has an anchor element to remove the facet and refresh the search results.

1,807 resources found for Young Adult ×, Fiction ×. 

.results-summary {
  @include results-summary;
}

NOTE: Maybe replace a tag with button?


<div class="results-summary">
  <span class="results-count">1,807</span> resources found <span class="facet">for
  <strong>Young Adult</strong>
  <a href="#" title="Remove Young Adult filter">×</a></span>,
  <strong>Fiction</strong>
  <a href="#" title="Remove Fiction filter">×</a></span>.
</div>
